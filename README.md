# Padrão Strategy — Estudo de Caso: Conta de Estacionamento

## Descrição do trabalho
Trabalho de Engenharia de Software II — Padrões de Projeto (GoF).
Estudo, implementação e comparativo do padrão **Strategy** usando um problema de cálculo de tarifas de estacionamento.

## Objetivos
- Demonstrar problema **sem o padrão** (solução rígida e dificil de manter).
- Demonstrar solução **com o padrão Strategy** (algoritmos encapsulados, maior flexibilidade).
- Apresentar análise de vantagens e desvantagens.
- Preparar apresentação (7–20 min) e código no Git.

## Padrão escolhido
**Strategy** — permite encapsular famílias de algoritmos, torná-los intercambiáveis e separar o código que decide qual algoritmo usar do próprio algoritmo.
